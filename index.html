Git Book

Gitコマンドの自分用メモ

git log

git log --all
全ブランチ
git log --name-status
変更したファイル名（あとで補足する）
git log -p <path>
パス（ファイル）を指定できる

git logは、コミットログが見れる。パラメーターつけて、見たい情報をしぼり込むと便利。
基本的には今いるブランチのログを見せるんだなぁ。
--graphというオプションも便利。ツリーで見れる。
でも、よりグラフィカルな表示ができるツールも便利。Sourcetreeやら、VSCodeの拡張やら(Git Graphが好きです)、tigやら。

git reset

git reset HEAD
ステージングを取り消す
git reset --hard HEAD^
直近のコミットを破棄して、ひとつ前のコミットが完了した状態に戻す

HEADのことをもっと理解しないと…

git show

git show コミットID
コミットを指定して、コミットの詳細を表示

git stash

git stash pop
スタッシュを適用して、スタッシュを削除するまで

git diff

git diff --cached
ステージ済みのdiffを見る
